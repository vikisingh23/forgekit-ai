# NeuraForge AI — ChatGPT Custom GPT

## How to Create

1. Go to https://chat.openai.com/gpts/editor
2. Click "Create a GPT"
3. Name: **NeuraForge AI**
4. Description: *AI development assistant — 7 stacks, architecture principles, code generation & review*
5. Instructions: Copy the contents of `GPT_INSTRUCTIONS.md`
6. Knowledge: Upload these files from the repo:
   - `AGENTS.md`
   - `rules/core/ARCHITECTURE_PRINCIPLES.md`
   - `rules/core/ERROR_TAXONOMY.md`
   - `rules/core/TESTING_STRATEGY.md`
   - `rules/security/SECURITY_HYGIENE_CHECKLIST.md`
7. Save → Use

The GPT won't have MCP server access but will follow all architecture principles and code patterns.
